# Support

## Getting Help

- **Bug reports** -- Open a [GitHub Issue](https://github.com/groupesti/laravel-ca-key/issues) using the bug report template. Include your PHP version (8.4+), Laravel version (12.x or 13.x), and package version.
- **Feature requests** -- Open a [GitHub Issue](https://github.com/groupesti/laravel-ca-key/issues) using the feature request template.
- **Questions and general discussion** -- Use [GitHub Discussions](https://github.com/groupesti/laravel-ca-key/discussions).

## Before Opening an Issue

1. Search existing issues and discussions to avoid duplicates.
2. Verify you are using a supported PHP version (8.4+) and Laravel version (12.x or 13.x).
3. Include a minimal reproduction case (code snippet or repository link).

## Important

GitHub Issues are **not** a general support forum. Questions posted as issues may be closed and redirected to Discussions.

## Security Vulnerabilities

Do not report security vulnerabilities through public issues. See [SECURITY.md](SECURITY.md) for instructions.
