# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

All participants in this project are expected to abide by its terms. Please read the full text at the link above.

## Reporting

If you experience or witness unacceptable behavior, please report it by emailing **security@groupesti.com**. All reports will be reviewed and handled confidentially.
